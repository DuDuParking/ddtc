目录说明
  • tools:项目生成发布工具
    • apache-ant-1.9.4:ant
    • jar:js压缩库
  • development:开发代码目录
    • golba_rs:公共资源
    • test:测试代码
    • userhtml:车主应用web目录
    • secenhtml:现场管理应用web
  • _publish:发布目录,执行ant生成
发布
  • 命令行跳转:cd tools/apache-ant-1.9.4/bin
  • 命令行执行:./ant -buildfile ../../../development/build.xml
  • _publish目录就是最新发布项目
